## Conversor de Temperaturas
Aplicación de consola en Java que convierte temperaturas entre grados Celsius y Fahrenheit.

### Características del Conversor
- Conversión bidireccional: Celsius ↔ Fahrenheit
- Interfaz de consola interactiva
- Proyecto estructurado con Maven
- Manejo de entrada de usuario

### Cómo Ejecutar el Conversor
1. Navegar al proyecto:
   ```
   cd ConversorTemperaturas
   ```
2. Compilar con Maven:
   ```
   mvn compile
   ```
3. Ejecutar:
   ```
   mvn exec:java -Dexec.mainClass="ejemplo.conversortemperaturas.TemperatureConverter"
   ```
